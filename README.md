# Taiwan Holy Young 2024 - Crypto Lab

## Deploy Challenge To CTFd

需要寫一個 `secret.py`

```py
SERVER='34.81.172.164'
CTFD='http://34.81.172.164'
USERNAME='...'
PASSWORD='...'
```

然後執行

```shell
./ctfd.py
```


---
## Deploy Challenge
> 需要執行者有執行 `docker` 的權限

- Start challenge
    ```shell
    ./deploy start
    ```
- Stop challenge
    ```shell
    ./deploy stop
    ```
