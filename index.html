課前準備指引

哈囉大家，我是這堂課的講師 Curious，這邊會詳細的跟大家講上課前需要做什麼準備，所以大家要仔細把這頁讀完喔！

Discord Invite Link 簡報

聯絡資訊：首先請大家加入這堂課的 Discord Server，不管是課前課中或課後有任何問題都可以在 Discord Server 問或是直接私訊我（我的帳號是 Curious）

先備知識：這堂課會需要大家用 Linux 和知道怎麼寫 Python（特別是關於 str 和 bytes 相關的操作），不需要非常非常熟練，所以還不會的同學趕快上網學習一下

環境安裝（for 所有不是 Apple Silicon Mac 的電腦）：

  1. 安裝 VirtualBox：可以參考 這篇 Guide 來安裝，我使用的 VirtualBox 版本是 7.0.18，如果可以的話盡量跟我用同一個版本
  2. 下載 ova：到 這裡 下載 ova
  3. 匯入 ova：可以參考 這篇 Guide 來匯入
  4. 確認一切正常：把剛剛匯入完的機器開機，帳號和密碼分別是 user/user，如果能正常登入的話那就一切正常

環境安裝（for Apple Silicon Mac）：我會建議盡可能去找一臺非 Apple Silicon 的電腦，因為 Apple Silicon Mac 對新手來說很不友善，環境安裝非常的麻煩，但如果你真的完全沒有非 Apple Silicon Mac 電腦的話，那再跟我說一下，我看看要怎麼幫你解決

最後最後，這份簡報 是我們這次上課會講到的所有內容，大家有興趣的話可以先看一下，另外也不要忘記在這個平台上註冊一個帳號，註冊後就可以看到上課會用到的題目，大家如果有會的題目也可以先打喔～
